Protect and secure your site from PHP Fatal errors.

== Description ==

Want you to protect your WordPress Site from Fatal Errors?

Here is our plugin which will protect your site from Fatal Errors.

**How does it work?**
A Fatal error is thrown from a plugin which you aren't aware of it. Our Plugin won't allow them to crash your site.
It will deactivate the plugin which causes the fatal error.

**How to configure?**
You don't want to do any configuration unless the plugin is able to write on your wp-config.
If the wp-config is not having the writing permission then you have to do it by manually.
For more details and reference refer this documentation <a rel="nofollow" href="https://moreaddons.com/setting-up-php-fatal-error-blocker/">Setting up PHP Fatal Error Blocker Pro</a>

**Pro Features**
 - **Notification**: Whenever a error is detected on your site, our plugin will send you a notification to the email that you configured.
 - **Error Levels**: In PHP lot of error levels are available. Detect all type of errors and run your WordPress Site without a crash.
 - **Error Logs**: Whenever a error is detected on your site, our plugin will capture the error and write them into a log which can be easily accessed.
 - **Support & Updates**: If any features added you will get the automatic update. And also you will have no Queue on your Support Requests.

For complete list of features and details, please visit <a rel="nofollow" href="https://moreaddons.com/downloads/php-fatal-error-blocker/">PHP Fatal Error Blocker Pro</a>

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Thats it! you can now configure the plugin.
